# Assignment Reminder System

A full-stack web application to help students track assignments and deadlines.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Spring Boot, JPA, Hibernate
- Database: MySQL
- Auth: JWT

## Features
- Role-based dashboards (Teacher / Student)
- Assignment creation & tracking
- Completion status
- Notifications & reminders

## Run Locally

### Backend
cd backend  
mvn spring-boot:run  

### Frontend
cd frontend  
npm install  
npm run dev
